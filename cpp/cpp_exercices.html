§01 — Variables & Arithmétique
01
Calculer l'aire d'un rectangle int, opérateurs arithmétiques
EASY ✓ ▶
Déclare deux variables int : largeur = 7 et hauteur = 4.
Calcule l'aire et affiche-la avec cout.

Résultat attendu :
// OUTPUT
Aire = 28
Aire d'un rectangle = largeur × hauteur.
Utilise int aire = largeur * hauteur; 
#include <iostream>
using namespace std;

int main() {
    int largeur = 7;
    int hauteur  = 4;
    int aire     = largeur * hauteur;
    cout << "Aire = " << aire << endl;
    return 0;
}
02
Conversion Celsius → Fahrenheit float, formule mathématique
EASY ✓ ▶
Déclare un float celsius = 100.0.
Convertis en Fahrenheit avec la formule : F = C × 9/5 + 32
Affiche le résultat.
// OUTPUT
100 C = 212 F
Attention : en C++, 9/5 avec des int donne 1 (division entière).
Écris 9.0/5.0 ou utilise un float. 
#include <iostream>
using namespace std;

int main() {
    float celsius    = 100.0;
    float fahrenheit = celsius * 9.0 / 5.0 + 32;
    cout << celsius << " C = " << fahrenheit << " F" << endl;
    return 0;
}
§02 — Conditions
03
Pair ou Impair if / else, opérateur modulo %
EASY ✓ ▶
Déclare un int n = 17.
Affiche "pair" si n est pair, "impair" sinon.
Teste avec plusieurs valeurs.
// OUTPUT pour n = 17
17 est impair
Le modulo % donne le reste de la division.
Si n % 2 == 0 → pair. Sinon → impair. 
#include <iostream>
using namespace std;

int main() {
    int n = 17;
    if (n % 2 == 0)
        cout << n << " est pair" << endl;
    else
        cout << n << " est impair" << endl;
    return 0;
}
04
Maximum de trois nombres if / else if / else
EASY ✓ ▶
Déclare trois int : a = 12, b = 47, c = 31.
Trouve et affiche le plus grand des trois.
// OUTPUT
Max = 47
Compare d'abord a et b pour trouver le plus grand des deux,
puis compare ce résultat avec c. 
#include <iostream>
using namespace std;

int main() {
    int a = 12, b = 47, c = 31;
    int max;
    if (a >= b && a >= c)
        max = a;
    else if (b >= a && b >= c)
        max = b;
    else
        max = c;
    cout << "Max = " << max << endl;
    return 0;
}
05
Mention au bac if / else if chaîné, opérateurs &&
MED ✓ ▶
Déclare un float note = 14.5.
Affiche la mention correspondante :
 • < 10 → "Recalé"
 • 10–12 → "Passable"
 • 12–14 → "Assez bien"
 • 14–16 → "Bien"
 • ≥ 16 → "Très bien"
// OUTPUT pour 14.5
Mention : Bien
Utilise des else if en cascade.
Pour vérifier un intervalle : note >= 12 && note < 14.
Ou plus simple : vérifie les seuils dans l'ordre avec juste else if (note >= X). 
#include <iostream>
using namespace std;

int main() {
    float note = 14.5;
    cout << "Mention : ";
    if      (note >= 16) cout << "Tres bien";
    else if (note >= 14) cout << "Bien";
    else if (note >= 12) cout << "Assez bien";
    else if (note >= 10) cout << "Passable";
    else                 cout << "Recale";
    cout << endl;
    return 0;
}

07
Somme de 1 à N boucle for, accumulateur
EASY ✓ ▶
Calcule la somme de tous les entiers de 1 à 100 avec une boucle.
Affiche le résultat. (Bonus : vérifie avec la formule N×(N+1)/2)
// OUTPUT
Somme = 5050
Crée une variable somme = 0 avant la boucle.
Dans la boucle : somme += i; (équivalent à somme = somme + i) 
#include <iostream>
using namespace std;

int main() {
    int somme = 0;
    for (int i = 1; i <= 100; i++) {
        somme += i;
    }
    cout << "Somme = " << somme << endl;
    return 0;
}
08
Triangle d'étoiles boucles imbriquées for
MED ✓ ▶
Affiche un triangle d'étoiles de 5 lignes.
Ligne 1 → 1 étoile, ligne 2 → 2 étoiles, etc.
// OUTPUT
*
**
***
****
*****
Tu as besoin de deux boucles : une pour les lignes (i de 1 à 5),
et une à l'intérieur pour afficher i étoiles.
N'oublie pas le endl après chaque ligne. 
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        for (int j = 0; j < i; j++) {
            cout << "*";
        }
        cout << endl;
    }
    return 0;
}
§04 — Fonctions
09
Fonction puissance fonctions, paramètres, return
MED ✓ ▶
Écris une fonction puissance(base, exposant) qui retourne baseexposant.
Teste avec 2⁸ et 3⁴.
// OUTPUT
2^8 = 256
3^4 = 81
La fonction reçoit deux int et retourne un int.
Utilise une boucle : multiplie base par lui-même exposant fois.
Commence avec un résultat = 1. 
#include <iostream>
using namespace std;

int puissance(int base, int exp) {
    int resultat = 1;
    for (int i = 0; i < exp; i++) {
        resultat *= base;
    }
    return resultat;
}

int main() {
    cout << "2^8 = " << puissance(2, 8) << endl;
    cout << "3^4 = " << puissance(3, 4) << endl;
    return 0;
}
10
FizzBuzz boucle + conditions + modulo — classique entretien
HARD ✓ ▶
Affiche les nombres de 1 à 30. Mais :
 • Multiple de 3 → affiche "Fizz"
 • Multiple de 5 → affiche "Buzz"
 • Multiple de 15 → affiche "FizzBuzz"
 • Sinon → affiche le nombre
// OUTPUT (extrait)
1 2 Fizz 4 Buzz Fizz 7 8 Fizz Buzz 11 Fizz 13 14 FizzBuzz ...
Vérifie le cas % 15 == 0 en premier (avant 3 et 5),
sinon tu afficheras "Fizz" au lieu de "FizzBuzz" pour les multiples de 15. 
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 30; i++) {
        if      (i % 15 == 0) cout << "FizzBuzz";
        else if (i % 3  == 0) cout << "Fizz";
        else if (i % 5  == 0) cout << "Buzz";
        else                   cout << i;
        cout << " ";
    }
    cout << endl;
    return 0;
}
